# Agent Instructions

These instructions are mandatory for all AI agents working on this project.

1. **Package Manager**: Always use `pnpm` (never `npm` or `yarn`) for installing dependencies and running scripts.
2. **Code Quality**:
   - Ship only high-quality, production-ready code.
   - Ensure code is clean, readable, and maintainable.
3. **Error Handling**:
   - Use `try/catch` blocks appropriately to handle errors gracefully.
   - Ensure robust error handling patterns are in place.
4. **Localization**:
   - Support both French and English versions for all content and features.
